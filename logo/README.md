## Logo Info
The text logo is generated using [toilet](http://caca.zoy.org/wiki/toilet):
```
toilet -f slant -F border "mkdev 3.x"
```

The svg is generated in inkscape with the font "Fira Code NerdFont Mono". The foreground
colour is CSS's "orange" named colour.
